# Manage Websites

Tools for managing existing websites in the workspace.

## Available tools
- `list_websites` — List all websites
- `get_website` — Get details for a specific website
- `update_website` — Update website name or description
- `delete_website` — Delete a website
- `update_website_settings` — Update website configuration
- `add_custom_domain` — Add a custom domain to a website
- `remove_custom_domain` — Remove a custom domain
- `get_website_analytics` — View website traffic and analytics
- `get_website_team` — View team members on a website

## When to use
- User asks to see their websites or a specific website
- User wants to change website settings, name, or domain
- User asks about website traffic or analytics
- User wants to manage team access to a website
